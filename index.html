Projects of mine

  • apache-log-replay: Small tool for replaying an access log on an apache server, interesting approach to try reproduce the traffic on a website and identify possible bottlenecks that depends on a particular user flow (even though HTTP is stateless and so should be your application!).
  • freq: Ridiculous application to be used with process pipes to count and display the number of lines the process has produced. You can, for example, roughly count the number of requests your webserver is receiving directly from the CLI. Or you can use it as an example on how Unix signals API work.
  • annotate: Interesting approach based on Stripe's documentation page to implement a sort of TL;DR system for learning or summarizing long contents.
  • timedbuff: An ought to be generic JS module to provide a timed buffer (list) where each item has a time-based lifetime. For example, when you want to keep only the calls that happened within 30 minutes on a callcenter application.
